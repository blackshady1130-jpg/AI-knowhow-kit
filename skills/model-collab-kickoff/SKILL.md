# Model Collaboration Kickoff

## Use When

- 用户要写合作 kickoff 文档或邮件。
- 任务涉及业务团队、模型团队、数据团队、评测团队协作。

## Required Sections

1. 目标：为什么做，解决什么问题
2. 数据：来源、格式、质量标准、规模预估
3. 训练：对应 pretrain/SFT/RLHF 哪一环
4. 评测：指标口径、基线、验收标准
5. 落地：ToB/ToC场景、交付方式、里程碑
6. 分工：谁负责什么，依赖谁

## Default Deliverables

- Kickoff 邮件正文（精炼版）
- 一页框架（可放PPT）
- 行动清单（owner + deadline）

## Failure Checks

- 只写愿景、不写验收标准 -> 不通过
- 只写模型能力、不写业务场景 -> 不通过
- 只写分工、不写依赖与里程碑 -> 不通过
