# CLAUDE ENTRY (v0.3)

在本仓库工作时，请按以下顺序加载：

1. `PROFILE_CORE.md` 与 `WORKING_PREFERENCES.md`
2. 涉及行业判断时，使用 `DOMAIN_KNOWHOW.md` 的框架
3. 涉及写作改写时，使用 `STYLE_GUIDE.md` 的语气规则
4. 对争议信息，可回查 `docs/EVIDENCE_INDEX.md` 与 `docs/sources/`

## 检索路由

- AI 行业扫描主题：先读 `AI_SCAN_RETRIEVAL.md` 并按其检索顺序作答。
- 资料补充/案例扩展：先读 `BOOKMARKS_RETRIEVAL.md`。
- 双库协同：按 `AI_SCAN_RETRIEVAL.md` 第 7 节执行。

## 明确禁止

- 过度讨好式表达。
- 不标边界的绝对化结论。
- 只给概念，不给落地路径。
