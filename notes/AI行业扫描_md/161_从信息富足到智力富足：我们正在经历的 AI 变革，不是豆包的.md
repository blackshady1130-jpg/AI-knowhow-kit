# 从信息富足到智力富足：我们正在经历的 AI 变革，不是豆包的模样

写给家人，也写给所有正在经历这场变化的人。春晚的豆包，耍武术的铁哥们儿，不是AI的真模样。一场真正的变革，已经悄然到来，带着清华学生水平的智力，和不眠不休的执行力，从智力需求最密集的工种开始，开启智力富足的新时代。

## 信息的富足

我还记得第一次上网的场景，好像也是在马年。

走进一条胡同，拐进一个小院，几十平米的房间里摆着不到10台电脑，所有人都在打游戏，似乎是红警。我跟老板说我要上网，她帮我设置好，我打开浏览器，输入从计算机培训书上背下来的 sina.com、sohu.com——那一刻，我觉得自己实现了知识自由。

后来第一次用手机，已经记不清是哪年了。表姐把翻盖手机给我玩，发出信息的瞬间，我仿佛和整个世界建立了一种神奇的连接。再后来，家里买了一部诺基亚，我第一次用彩信发送了一张照片，好奇心被大大满足。

再后来就用上了搜索引擎——我的任何好奇，终于有了答案。大树为什么向上生长？风从哪里来，又怎样停下来？

大学时，互联网通畅无比，搜狗浏览器还可以科学上网，“免费版”软件和资源满天飞，不花钱要啥有啥。我和舍友在联想 ThinkPad 笔记本上把奇奇怪怪的大软件都装一遍，比如Adobe全家桶，Maya, Matlab, Maple, 也会去图书馆 TP 书架区把计算机的书翻一遍，在塞班手机上把能装的应用玩一遍，每类软件，都要找到最好的那一个。这个习惯保持到现在，当我要用一个户外路线App，就把主要的App都调研一遍，找到最好的那个。

互联网和移动互联网，给了我们信息的富足。

## 智力的富足

从“聊天机器人”，到“深度研究”，再到“干活机器人”。

### 聊天机器人

2022年底，ChatGPT1上线那天——我的任何问题，终于有了解答。对话这种形式，就像和人交流。现在想想，那天我们都在做图灵测试。

这跟搜索引擎不一样。搜索引擎给你十个链接，你自己翻；AI 能理解你的问题，针对性地给你答案。

### 深度研究

但聊天只是开始。我日常工作里有一件很花时间的事：调研。如果要调研一个研究领域，我会从 Google、LinkedIn、GitHub、实验室主页、个人网站、论文里搜集信息，拼出人与人之间的合作关系，推断论文背后的故事，以及这个领域的来龙去脉。这是基本功，但也需要训练，不仅要搜集信息，还要能理解信息，并建立关联，边推断边搜集，很花时间。所以只有一些重要的事情，才会这么干。

2025年初，ChatGPT 推出的 Deep Research2（深度研究）功能干的就是这个事，它可以自动搜索几十甚至上百个网页，交叉比对信息，最后给你一份带引用的完整报告。我以前花一天才能拼出来的东西，它10分钟就能完成。这个阶段，AI 替代的不是我不会做的事，而是我很会做、但要花大量时间的事。

### 干活机器人

2025年年中，Claude Code3来了，这是一个代码开发工具，但通过对话完成开发，这种开发方式被称为“vibe coding”（氛围编程）。整个2025年我已经没有手写过一行代码了，先是代码IDE Cursor4干掉了手动编辑，然后Claude Code干掉了IDE，软件开发不需要交互界面了，也不打字，我说话，它干活。讲几个最近遇到的 AGI 时刻。

最近我们开发了一个语音输入法产品，和第一个用户交流后，我把十个问题和截图丢给 Claude Code，它自己读了整个代码库，把反馈整理成issues（问题描述），排好优先级，从最容易见效的问题开始解决。涉及到效果优化，它自己去数据库里找测试数据，建立测试集，制定一套评估标准，然后改一轮代码、测一轮效果，再改、再测，最后提交代码，合并，关闭issues，发布。不到一天，第二个版本就上线了。

整个过程中，它同时扮演了研究员、工程师、项目经理和测试员四个角色。我只需要在关键节点给方向和确认，就像管理团队时一样。

还有一次，我让它排查一个产品问题。常规的日志分析走不通——数据库里的信息是加密的，看不到。它自己换了思路：写了一个监听脚本，加了调试日志，让我在软件里做一些操作，它从日志里分析蛛丝马迹，很快定位了问题的根源。它在完成我的任务，而不是执行我的指令。

还有一次更有意思。干活过程中，我以为它搞错了，打断它，跟它说你这里不对。结果它说：我没错，你自己看看。然后给我打开了那个文件。我一看——确实是我搞错了。

我不禁感叹这个新模型的智商好高（几个月以前总觉得它不太靠谱，只让它干杂活，难的活交给另一个AI干，现在不一样了），看着它干活的过程就特别享受，只要在合适的时候指导一下就行了。我指导过许多研发实习生，清华最优秀的实习生大概就是这个水平，但速度要慢很多，可能要2天或者一周，而且要花很多时间讨论。同样的事情，现在的AI只要半小时，而且和AI交互的过程中，沉淀下来的经验，它会记住，会复用，永远跟着项目走，不会离职。

如果说互联网给了我们信息的富足，那么**AI 给了我们智力的富足**。它接管了大量需要调研、思考、执行的工作。我们获得的不只是信息，而是智力活动本身的富足。

过去在企业里，人是最大的杠杆——招到一个优秀的人，能撬动一整个项目。这一点正在改变。

在智力劳动密集的事情上，AI 正在成为最大的杠杆。

## Agent：不只是一个聪明的脑子

前面讲的这些——自己排查问题、自己建测试集、还会跟你争论——靠一个只会聊天的模型是做不到的。ChatGPT、DeepSeek5这些大语言模型，本质上只是一个聪明的脑子，叫基础模型。光有脑子不够，2025 年真正的变化是：**Agent。**

### 什么是 Agent

在人工智能里，有一个经典的概念：一个 agent 观察环境，做出决策，然后执行动作，作用于环境；环境发生变化，agent 再观察反馈，再决策，再行动——如此循环。DeepMind 联合创始人 Shane Legg 和 Marcus Hutter 在 2007 年给智能下了一个定义6：**智能衡量的是一个 agent 在各种各样的环境中达成目标的通用能力。**

换句话说，智能衡量的是 Agent，不是基础模型。

基础大语言模型只是 agent 的“脑子”——它有知识、有推理能力，但它是被动的，你问一句它答一句。Agent 不一样，它能主动跟环境交互：观察、思考、行动、再观察。这也是今天 Claude Code 在做的事情：基础模型提供智力，agent 的循环机制让它能够一步一步地完成复杂的任务。

还有一件事让 agent 变得更强大：**工具。**

人类一直在用工具放大自己的能力。用计算器替代心算，用自行车、汽车让自己移动得更快，用电脑处理信息。今天的 AI agent 也一样。在数字世界里，Claude Code 本质上是在用我们的电脑：它会用搜索引擎查资料，用查询指令搜索代码，用多模态模型读取图像，写脚本来执行更复杂的操作——这些都是工具。

还有一点很像人：**记忆。**人不会把所有信息都存在脑子里，而是写成书、记到笔记上，形成长期记忆。以后需要解决问题的时候，再从书和笔记里查找相关信息，然后做决策。今天的 agent 也一样——它的“脑容量”（context length）是有限的，但它也可以记笔记、存储文档，形成自己的长期记忆。对人来说，纸和笔是工具；对 agent 来说，它也有自己的“纸和笔”。

所以完整的图景是这样的：先有智力（基础模型），有了智力之后使用工具，使用工具再跟环境交互。**它是一个有记忆、有推理能力、又会使用工具的 agent。**这是一个跟基础大语言模型不同的新范式。基础模型是地基，agent 是建在地基上的、能真正干活的东西。

### AI 已经成了一个 team

看个真实案例。我们用 AI agent 开发了一款语音输入法——Flow 输入法7(flowkeyboard.com)。一周做出原型，两周打磨到好用，主要靠 Claude Code 和 Codex8。上线需要打包上架 App Store、建网站、做宣传材料——App Store 的分发完全通过 API 实现，不需要在网页上点来点去；Logo、商店截图、各种素材直接用 Claude Code 写代码生成；网站也是 Claude Code 生成的，运维也全部交给它。人需要做的，就是给它方向和API授权。

从设计、开发到上线，所有事情基本都通过和 AI 协作来完成。**AI 已经成了一个 team。**它同时扮演产品经理、设计师、前后端开发工程师、算法工程师、运维工程师等多个角色。做 UI 设计的时候，派出偏前端设计的 agent；做性能优化的时候，派出偏研究的 agent。整个项目里，除了人负责方向和决策，其他工作都可以让不同角色的 agent 来做。这就是我们现在的开发方式。

现在这款输入法已经成了我每天在手机和电脑上打字时最常用的工具，到目前为止我已经用它输入了六万字。

## 人的价值

说到这儿，一个更大的问题来了。我们应该会经历很长一段时间的人+AI的混合体，但许多职业将逐渐彻底被AI接管。AI 这么厉害，那人的价值在哪里？

### 那些 AI 给不了的瞬间

把鸡蛋一颗一颗放进冰箱的蛋槽里，这个动作太简单了，我做起来毫不费力。但我突然想：如果让机器人来做呢？手、眼、脑的协调，力度的拿捏，空间的判断——今天的机器人恐怕做不到。切菜的时候看到番茄新不新鲜，炒菜的时候闻到香味，洗碗的时候把碗筷一个个放进洗碗机——这些我们习以为常的动作，复杂性其实极高。人能以这么灵巧的方式存在，真是一个奇迹。

2025年我参加了八场越野跑比赛，全都完赛了，有些是一百公里，要跑一个晚上，有一场是一百英里，要跑过两个夜晚。夜里戴着头灯跑在山脊上，只听到自己的呼吸和脚下碎石的声音；一回头，巨大的满月正升起来。身体在变化，天气在变化，你会感受到专注，感受到快乐，也感受到痛苦。一群人在同一时间做同一件事，有点像某种神奇的仪式。这不是为了生存，就是为了体验本身。

我就在想：假如机器人也能越野跑，它会想做这件事吗？它会从中感受到快乐吗？应该不会。爱因斯坦是最聪明的人之一，但他会从越野跑中获得快乐吗？不一定。每个人做事情的动机是不一样的。聪明不是一切。

除夕那天晚上，去一个朋友家聚会。朋友是大厨，做了很多好吃的菜。一起包饺子，看了一部日本电影《小森林》9，电影里全是美食。吃年夜饭时，三只小狗一只小猫在旁边陪着我们，小狗口水流得哗啦哗啦的。傍晚日落，蓝调时刻，香喷喷的饭菜配上窗外的暮色，拿出相机记录下这一刻。

这些事情跟智力有关吗？

### 超越智力的东西

人和人之间、人和宠物之间、人和自然之间，有很多很多的连接。这种连接跟 AI 毫无关系。即使有一个比爱因斯坦还聪明的 AI，它可以干各种各样的事情，但这些东西是无法替代的。每个人都很独特，你和别人之间的连接很独特，你的感受也很独特。

小猫小狗也有智能，虽然它们不懂语言。人真的比动物高贵吗？人对宠物的爱，和对子女的爱，在很多方面极其相似。即使面对牲畜，人们有时候也超越了功利的目的。小时候，爸妈养猪，有一头小猪受伤了，他们就把它单独圈养起来，每天细心照料。后来这头猪不仅康复了，还长成了一头非常健壮的成年猪。当然它卖出了好价钱，让人开心，但照料它本身——看着它一天天好起来——也是一件值得开心的事情。人跟牛之间的关系10，藏民和牦牛之间的关系11，哈萨克人和马之间的关系12，大抵如此。

**所有这些跟智力无关的东西，反而是我们作为人最独特的。**

人会好奇，会期待——李健说他保持生活热情的方式，是对一些小小的事情保持期待，比如期待一把定制的吉他13。人会为大自然的一个瞬间感动，会分泌激素，会感到恐惧，会有情绪，会说“不”。这些感受，AI 都没有。

“聪明”从来不是最重要的品质，今天更是如此。当 AI 比人更聪明的时候，那些我们以前不够重视的东西——同理心、敏感、好奇、品味，人与人之间的关系，人和其他生命之间的关系——这些会变得前所未有地重要。

人类学家泰勒在一百多年前给文明下过一个定义14：文明是人作为社会成员所习得的一切——知识、信仰、艺术、道德、风俗。注意，不是个人的聪明才智，而是人与人之间共同积累下来的东西。历史学家布罗代尔研究文明的方式更直接15：他不看帝王将相，他看人们怎么吃饭、怎么穿衣、怎么生活在一起——文明的本质藏在日常生活的结构里。

AI 可以让我们更聪明，但文明这件事，只能靠我们自己往下传。

## 结语

2026年，新时代已经开启，它带来的冲击将远超电话、电脑、手机对人类文明进程的影响，人类曾经最引以为豪的智力，将在2026年变得富足，那些清华姚班的学霸传说，北大青鸟也将唾手可得。

总结一下新时代的生存法则。

**第一，了解真相。**AI 发展到什么程度了，现在只有极少数人有切身体会。这些人正在利用 AI 建立巨大的优势。是的，真相可能带来焦虑，但不了解真相的代价更大。

**第二，用起来。**过去想要用好计算机，需要掌握编程技术，开发软件。但今天的计算机能理解自然语言，每个人在自己领域都有独特的认知和经验，都有自己的想法和好奇心。你可以用自然语言告诉 AI 你想做什么、想创造什么，然后一起把它做出来。

**第三，重新理解价值。**AI 是工具，让它帮我们把时间解放出来，去探索对人更有意义的事情。不要把智力当作衡量人的唯一标准——当 AI 比人更聪明的这一天，我们需要的不是更高的智商，而是更丰富的人性。

去探索你到底想要什么。关心自己，关心身边的人，关心其他生命，关心物理世界。人的存在本身，就是有意义的。

那个走进胡同网吧的少年，好奇心从未停过。只是现在，好奇心有了翅膀。
