# 深度｜Anthropic CEO：AI行业的盈利本质上源于对市场需求的低估，而亏损则因为高估了需求，2030年AI行业营收将跃迁至万亿美元级

Dwarkesh Patel | Z Potentials

本次访谈于 2026 年 2 月 14 日开展，围绕 AI 技术指数增长尾声、模型缩放假说、RL 发展等核心问题展开。

## 核心观点

- 底层技术的指数式发展整体符合预期，各领域技术前沿发展虽不均衡但大体走向无偏差，令人意外的是，无论行业内外，人们都未意识到我们正如此接近指数增长的尾声
- 自 2017 年起核心缩放假说从未改变，真正推动技术发展的核心要素仅有七个，如今 RL 领域也展现出和预训练相同的缩放规律
- 有 90% 的信心认为十年内数据中心将诞生堪比一个国家的天才智能体，1 到 2 年内模型就能实现端到端的代码开发
- AI 技术的发展和向经济领域的渗透会极其迅速但并非一蹴而就，存在客观限制
- AI 行业盈利的核心是算力需求预判，盈利源于需求低估，亏损则因需求高估
- API 商业模式仍具长期生命力，同时将诞生按结果付费等多元模式

## 技术发展核心认知

Dario 认为核心变化在于底层技术的指数式发展，前后误差也就一两年。模型从堪比聪明的高中生，到进阶为优秀的大学生，再到开始涉足博士阶段。代码领域的发展甚至超出了这一阶段。

但公众并未意识到我们正如此接近指数增长的尾声。"无论是行业内还是行业外的人，还在为那些老生常谈的热门政治议题争论不休，却对身边即将到来的技术拐点视若无睹。"

## 缩放假说

七个核心要素：预训练、后训练、RL from Human Feedback、推理时间计算(Chain-of-Thought)、工具使用、Agent 化和长上下文窗口。这些要素并非每一个都直接对应预训练里那种原始的 Scaling Law，但它们共同构成了一条汇聚的指数增长曲线。

## 经济渗透

AI 向经济渗透存在四个客观限制：可靠性限制、延迟限制、信任建立限制和人类惰性限制。"但是 AI 渗透经济的速度仍然会非常快。2025 年 AI 行业的全球营收大约是 100 亿美元，到 2030 年达到 1000 亿美元是高度可预期的，如果 AI 真的做到了我们认为它能做到的事情，从 1000 亿美元到 1 万亿美元的飞跃也只是时间问题。"

## 商业模式

"从根本上说，盈利源于你低估了需求，而亏损源于你高估了需求。这和传统行业完全不同。"

API 模式仍具长期生命力，"不是说 API 不好用，而是说 API 更像是食品的原材料而不是最终成品"。未来会有更多按结果付费的模式出现。

## 关于开源

将开源权重模型比作核武器类比是不恰当的，"核武器一旦流出，后果不可逆转。但是模型不一样，模型的价值是会贬值的。"
