# 奇绩创坛路演复盘：什么样的创始人、哪些行业，更容易拿到投资？ - 智源社区

![](https://simg.baai.ac.cn/hub-detail/671c81583f20876966409d5323e275781733575801958.webp)

上周日~果壳儿~来到了[奇绩创坛2024年秋季创业营路演日的现场](https://mp.weixin.qq.com/s?__biz=Mzg5NTc0MjgwMw==&mid=2247512153&idx=1&sn=905a5cd307234d5e3bfc5800320ac2c6&scene=21#wechat_redirect)，奇绩创坛投资项目作为早期资本关注赛道的风向标，具备前沿科技创新的前瞻性，此期有60家由奇绩创坛投资并加速的早期项目参展并上台路演，让我们一起来看看有哪些牛人、新人做了什么新事儿～

## **01** 

## **60个项目标签大数据分布**

-   项目整体录取率依旧约1%

-   分赛道来看，主要集中在**人工智能及具身智能相关**的算法、数据、行业/通用场景应用领域，与人工智能技术无关的项目仅3个。其中，非去重口径的大模型项目49家、多模态项目28家、数据项目24家，具身智能14家

-   人工智能项目按照细分方向划分，硬件解决方案、to C应用、算法项目最多，分别占比28%、18%和18%，其中to B硬件解决方案17个、to C应用11个、算法11个，其余还包括软件解决方案9个、数据9个项目

-   项目按照应用场景划分，**to B软件及机器人软硬件项目最多**，to B软件项目～20个，机器人软硬件项目～20个，其余为to C软件～14个、泛脑科学3个、to C消费电子1个、新材料1个

## **02** 

## **创始人画像分析**

-   作为侧重看“人”、看“人的经验”和“做的事”匹配度的早期投资，60个项目的创始人介绍中，**高频提及关键词包括连续创业者、大厂离职创业、清华系、海外长期生活经验、lab科学家、原生全球化公司**

-   60个项目中，创始人多为90后，平均年龄29岁，其中64%为硕士及以上学历，12%为女性创始人

-   分创始人主要标签来看，**科学家和大厂背景的创始人占比最多**，分别约35%和25%，其中科学家22位、大厂/操盘手15位、年轻人8位、连续创业者7位，其余还包括投资人转型、小天才、李泽湘系等

-   从教育背景来看，**最高学历为国内院校背景的创始人38位，占比～60%**；海外院校背景的创始人22位，占比～40%。此外，创始人休学创业的项目～10个，占比～20%

## **04** 

## **软件应用分析**

分To B和To C的软件应用，整体来看To B领域主要涉及垂直行业场景和AI安全相关的解决方案，早期项目多强调切入场景选择和数据积累价值；To C则聚焦在内容创作工具、个人知识管理、游戏、社交等领域，在切入场景和玩法设计上有一定创新性，属于老想法下的新模式。分享几点findings，也echo了果壳儿之前在 [纪实 | 又打飞的跑了一趟硅谷的新鲜事（万字感想）](https://mp.weixin.qq.com/s?__biz=MzkyNTcyMDYyMA==&mid=2247483956&idx=1&sn=479312b87a7d54d091eb882cc73781bf&scene=21#wechat_redirect) 文章中提到的软件出海及SaaS服务化趋势相关：

1）**出海项目超一半**：海外主体注册+做海外市场的华人创业项目占比显著增加

2）**AI服务化的商业模式**：软件SaaS项目不再强调产品化，而是+AI整套解决方案如工具平台、agent服务等，卖AI服务，而不只是AI产品

3）**年轻人创业有passion driven**：软件项目中不乏连续创业者、休学创业、长期海外留学经历的标签，能看到华人95后、甚至00后对AI领域的passion、基于兴趣的内测群和自发传道者、思维和交流上的openness、产品和市场的国际化等等趋势

4）延伸到软件领域投资的大范畴来看，跟踪了一些软件上市公司，判断是今年二级市场海外头部SaaS公司有+AI产品/服务的整体盈利有提升，**SaaS板块估值上涨**；在降息及全球科技竞赛的宏观大环境下，AI/软件赛道的并购市场持续活跃

## **05** 

## **硬件产品分析**

1）**关于机器人的硬件项目很多**：包括导盲机器人、园林机器人、工业机器人等，且与之相关也有好些仿真的项目。这是一个长坡厚雪的赛道，而且在各种类型的硬件项目里机器人背景的技术联创更是非常普遍。

2）**涌现了几个教育类别的GenAI硬件公司**，均是针对利用寓教于学的新“教具”，据说市场反馈很佳。

3）**二次元陪伴硬件**入选了两个公司，谷子经济的需求确实旺盛。

4）此外还有一些类别的项目，例如电动船、脑波调节可穿戴（治疗失眠）、AI眼镜，项目各有千秋。

-   项目简介：Neurobo提供了一套「养谷」平台，赋予「谷子」等二次元周边与物理世界交互的能力。平台结合 LLM 与现实场景数据，让每一个二次元用户将任意「谷子」变为随身相伴，随时触达的实体情感伴侣

-   团队背景：清华人机交互团队，全部在日本留学过

-   三个关键词：二次元，实体陪伴，多模态能力

-   亮点：养谷子平台：实现自己捏模块+硬件模块+训练模块。能形成一个虚拟形象真正进入物理世界成为深度情感陪伴。这在今日二次元爆发的时代，用户喜爱的静态IP加上多模态能力可能产生巨大需求

**6、汤问致新**

-   一句话介绍：打造最还原、最智能的伙伴机器人

-   项目简介：汤问致新机器人科技，致力于打造最还原、最智能的消费级具身机器人伙伴，基于交互机器人方向领先的机器人+AI 技术与团队，实现桌面级机器人在外观、运动、交互、智能等方面的突破，使机器人真正进入用户的三次元世界成为他们的贴心伙伴与智能终端。前期公司通过与各 IP 方达成授权合作，瞄准 IP 自身粉丝群体，再进一步推出基于自主 IP 的 C 端机器人终端产品

-   团队背景：哈工大机器人+清华社科

-   关键词：手办机器人、阶段性开发

-   亮点：已获得B端订单，将推出自己的C端IP；桌面级的小手办机器人将具有广阔的C端市场；自研微型小回差电机系统与刚柔结合设计，兼顾机器人的精致外观与动态运动，前段时间出来的小版本的optimus大获好评，相信与手办结合是一个大的市场
