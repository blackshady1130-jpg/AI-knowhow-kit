# 大模型未来不确定的下一步

让市场提心吊胆的一周开始了。七月底、八月初，微软、Meta、苹果与亚马逊，将依次披露最新一季财报。上周，特斯拉与谷歌财报发布，让市场对巨头在短期内继续引领大模型迅速进展的信心摇摇欲坠，拖拽纳斯达克100指数向下，创造了ChatGPT发布以来最大日跌幅。

更深层的担忧，来自对扩展定律（scaling law）指引的大模型的未来的不确定。OpenAI的ChatGPT激发了市场对大模型前景的狂热想象力，它的GPT-4与Sora又推高了期待。但近半年来，OpenAI的步伐明显慢了下来，GPT-5迟迟没有发布，Sora尚未规模试用，[**GPT-4o语音版**](http://mp.weixin.qq.com/s?__biz=MzkzMzI2ODAyNQ==&mid=2247488698&idx=1&sn=0d35b6af520f7b67d59302559b950320&chksm=c24e4becf539c2fabd7039e8cac06cc5517becddc9af9ce50b70af830c4897bdc68beb95c490&scene=21#wechat_redirect)跳票，只是画了一张[**通往AGI的路线图**](http://mp.weixin.qq.com/s?__biz=MzkzMzI2ODAyNQ==&mid=2247488954&idx=1&sn=f00c700099ad7e0c93469943818dc4c7&chksm=c24e4aecf539c3fadb6f7540319a912d5ebb399585948186a2381873cc88f5f8641d7009b6fb&scene=21#wechat_redirect)安抚市场。这家生成式AI时代最亮眼的巨星，在微软这棵摇钱树下，年度开支[**缺口仍然高达50亿美元**](http://mp.weixin.qq.com/s?__biz=MzkzMzI2ODAyNQ==&mid=2247489029&idx=1&sn=ffa5e4455479747cb8edcb89a485a61f&chksm=c24e4953f539c04571c59f86df133a82f08f52791cca99be48e6170f2c73f883b06a1110fadc&scene=21#wechat_redirect)；行业整体缺口已经是一个“6000亿美元问题”了。市场一直在寻找生成式AI的杀手级应用，同时开始转向端侧部署小模型。

**前沿开始减速**

OpenAI的问题是扩展定律的问题。它短期内有望继续生效，但速度开始减缓，首先表现在闭源大模型天花板效应的出现，被开源模型追平。Meta最近发布的Llama 3.1 405B，是[**第一个前沿级别的开源模型**](http://mp.weixin.qq.com/s?__biz=MzkzMzI2ODAyNQ==&mid=2247489014&idx=1&sn=2b9d05a50cbe84c89d72d48a0ef6179d&chksm=c24e4aa0f539c3b69ae46d033938a845a7d7fdaf94500ab978fa1ac3210c1951429163a30bde&scene=21#wechat_redirect)，已经能与GPT-4和Claude 3.5相媲美。去年，性能最强的开源模型Llama 2，表现只能齐平上一代的闭源模型。

闭源模型遇到的问题，开源模型也会遇到。但开源一直是软件、互联网与人工智能发展的基石。从MMLU指标来看，开源模型并没有陡然提速，而是得益于社区共同创新的生态，让其以高于闭源模型的速度稳定迭代。扎克伯格相信开源大模型可以安全地为每个人创造最大经济机会。

开源模型对闭源大模型的反超，让市场担忧巨头构筑于后者之上的商业逻辑，随着性能提升曲线趋缓而逐步瓦解。越来越多开发者可以选择合适的开源模型训练、微调与蒸馏出最适合自己业务与市场的产品。现金流循环枯竭，将进一步影响科技巨头继续引领大模型创新的能力与魄力。

开源模型及其商业模式，也将断绝美国科技巨头垄断全球市场的可能性。在这一领域，中国正在迅速赶上来。去年，零一万物还深陷套壳争议中，6个月后就交出了排名前列的模型；今年，深陷争议的轮到了美国斯坦福大学团队的Llama 3-V，它套壳了中国团队的大模型。Hugging Face的CEO克莱门特·德朗格（Clément Delangue）担心如果中国的开源技术成为黄金标准，对美国将是越来越大的挑战。中国的AI应用比大模型更有竞争力，在美国市场上，今年上半年下载量最大的移动AI应用中，前五的就有两款来自中国。

**数据不确定性**

科技巨头推动扩展定律继续生效，遇到了多重不确定性。数据的匮乏使得大模型扩展定律不再那么确定。Epoch AI最新预测，大模型将在2028年耗尽互联网上的所有公共人类文本数据。媒体报道OpenAI训练GPT-5就遭遇文本数据不足，也考虑过使用视频转录出的文本。这一堵“数据墙”也推高了高质量数据的使用成本，以避免稀缺的算力浪费在低质数据上，这让Scale AI这样的标注公司的估值飙升至138亿美元。

除了公共人类文本数据，在物理世界与数字世界所构成的现实世界之上，还存在视频数据、工业数据、社交数据与合成数据等。但工业数据与社交数据隐私与安全属性较强，前者还长期[**缺乏有效的数据治理**](http://mp.weixin.qq.com/s?__biz=MzkzMzI2ODAyNQ==&mid=2247488892&idx=1&sn=79ee0ccfe7d6374c3df3951708e68322&chksm=c24e4a2af539c33c7311cd036abcfc03fabe4ae0a0a3dd482803bc6ef5d9b9d3e977aca11ff7&scene=21#wechat_redirect)；合成数据在实际应用中也会“污染”模型。Meta称，基于开源模型的微调等，在一定程度上保障了企业合理利用私有数据。

大模型企业继续游走在“数据墙”的灰色地带，但免费使用数据的时代已经逝去。Anthropic违规爬虫抓取数据，导致网站带宽成本急剧上升，迫使不少网站修改规则或一禁了之。OpenAI[**与《纽约时报》的官司**](http://mp.weixin.qq.com/s?__biz=MzkzMzI2ODAyNQ==&mid=2247487963&idx=1&sn=004b4783244ed661529706d1581094db&chksm=c24e4e8df539c79b053b3866b3970f15d48fcc9e3bd49fd8fc355bb99d46d92e9e3f4c9167c5&scene=21#wechat_redirect)仍在进行中，最近几个月开始与Reddit、《大西洋月刊》和《金融时报》等出版商和内容提供商达成协议，采购付费墙内的数据。上周，社交媒体X的用户发现，他们的帖子以及与聊天机器人Grok的互动，在自己没有明确同意的情况下，都被“选中”用于“训练和微调”xAI的系统；而要修改这项设置，只能在桌面端，而不是更常用的移动端。

合成数据是新的方向，但更重要的是找到正确使用合成数据的方法。长期以来，部分研究者始终怀疑合成数据能否代表人类及其经验的广度，以及它能否被用来超越现有的最佳模型。《自然》杂志最新论文称，不加选择地使用合成数据，终将导向模型崩溃。Meta也证实Llama 3.1 405B用自己生成的数据训练，不仅无益，甚至可能有害。而且，如今生成式AI编造“合成”的文本已经充斥于互联网，污染了日益不足的人类公共文本数据资源。

**算力不确定性**

训练数据、参数规模通过算力兑现扩展定律。如今算力瓶颈也是一大不确定性。超大规模的算力部署意味着高昂的资本开支，进一步扩大大模型行业现已严峻的开支缺口。Epoch AI统计了多个大模型的训练成本构成，发现除了研发团队成本，AI芯片仅占其中的20%-30%，服务器其他关键IT组件、集群级互联成本与训练期间的能源成本合计占了30%-40%。在没有需求的情况下，算力支出不会无限制增长下去，它受公司预算与股东利益的约束。同时，这也意味着芯片产能问题，将从GPU扩展到更先进HBM内存与网络组件等。

将GPU芯片等转变为超大算力集群的算力，本身还是一项从未涉足过的困难的工程。更大的集群意味着单一的数据并行（需要高内存）、张量并行（需要高带宽、低延迟）或流水线并行都不再可行，需要结合各类并行方式，使用全新的网络拓扑设计与网络设备部署，甚至机架布局。

这样全新的庞大的工程，还面临巨大的可靠性挑战。各类芯片或组件的意外卡死或故障，乃至过热，都会导致训练效率下降甚至中断。Llama 3.1一共训练了54天，发生了419次组件故障，平均每3小时一次。这还只是16384个H100集群，马斯克xAI新搭建的10万H100全液冷集群，实际运行恐怕也不会轻松；OpenAI的下一个超算集群可能由10万块GB200组成。

计算的本质是能源的处理形式，能源将约束算力扩展。超大规模算力集群意味着用电量激增，美国正在面临[**人工智能带来的“电力危机”**](http://mp.weixin.qq.com/s?__biz=MzkzMzI2ODAyNQ==&mid=2247488809&idx=1&sn=3284ee9d64ae94cd2331311b1f7caf16&chksm=c24e4a7ff539c369ad2e5130be2d2663173ca4f54b572be0d89e6f39aaa904222308fa2b71c9&scene=21#wechat_redirect)，未来AI算力集群甚至占据美国用电量的10%以上。电力供应、绿电上网、电网容纳的进展，都将约束科技巨头研发更先进大模型的进度；它们[**对碳中和目标的承诺**](http://mp.weixin.qq.com/s?__biz=MzkzMzI2ODAyNQ==&mid=2247488944&idx=1&sn=97d3da841044a51b8da9e05a22c27e56&chksm=c24e4ae6f539c3f01eaa79e014e3e7bfd7cb5f2fc6f782ba1b8b34dee37727e3dba4f425e2db&scene=21#wechat_redirect)，也是一个稍显软性的约束。

同样，将能源转变为超大算力集群的算力，也是一项从未涉足过的困难的工程。Meta在训练Llama 3.1时就发现，随着算力集群的增大，环境因素与功耗剧烈波动对能源管理的挑战越来越大。日夜环境温度变化，都会改变GPU的动态电压，对吞吐量的影响在1%-2%之间；大量GPU在相近时点等待通信等行为，将使得算力集群功耗瞬间波动数十兆瓦。电力系统技术创新将制约算力部署规模。

**监管不确定性**

第三大不确定性来自监管。美国总统大选结果，将左右对科技巨头尤其是人工智能的监管政策，七巨头都在积极游说，试图将监管框架塑造成适合自己的模样。目前大量涌现的虚假内容，尤其是大选相关内容，也将影响监管决策。

美国即将走出拜登政府对人工智能的监管框架。今年来，联邦贸易委员会（FTC）主席莉娜·汗（Lina Khan）主持了对科技巨头的垄断调查，要求微软、OpenAI、亚马逊、谷歌与Anthropic披露交易细节；还与司法部达成协议，调查英伟达等的垄断嫌疑；她支持开源模型，还全面禁止竞业协议，支持人才自由流动。

特朗普对人工智能言之甚少，但他提拔的万斯，主张肢解谷歌这样的巨头，也支持[**对巨头发起反垄断调查**](http://mp.weixin.qq.com/s?__biz=MzkzMzI2ODAyNQ==&mid=2247488997&idx=1&sn=0a12c69d92929a376fdba9e802695fa9&chksm=c24e4ab3f539c3a56a883d5da4d69ce9680ba01bec94e9752ea3ceb5ff00685a338c6a762afe&scene=21#wechat_redirect)。而LinkedIn联合创始人里德·霍夫曼（Reid Hoffman）与IAC主席巴里·迪勒（Barry Diller），支持副总统哈里斯（Harris）在上任后换掉“没有帮助美国的”莉娜·汗。加州也开始拟定自己的监管法案，这也将影响对科技巨头对大模型的探索方向。

此外，欧盟的《人工智能法案》将在欧盟范围内正式生效，并将在未来三年分阶段实施，结合《通用数据保护条例》（GDPR）与《数字服务法案》(DSA)，将进一步约束科技巨头在数据等领域的行为。每个主权国家都在争夺全球人工智能监管的话语权。

人们仍在寻求更强大的计算技术与能源技术，这是让扩展定律继续生效的可能路径。成立于2021年的量子计算企业Quantinuum，计划以100亿美元的估值，最快明年上市。微软与OpenAI都下注了核聚变。

研究人员与科技巨头也在探索Transformer架构以外的可能性——这一架构仍然缺少灵活性，对每个token使用等量的算力，意义不大。

技术是在应用中成熟，并且柳暗花明的。技术并不会停下来，可能把现有的技术用得更便宜和更广泛，有时候更加重要，人形机器人与自动驾驶的商业落地也在加速推进。
