# AI 应用和智能体工厂

![](https://ai.azure.com/assets/favicon-black-7afa0dd6.svg)Microsoft Foundry

## ![Microsoft Foundry](https://ai.azure.com/assets/MicrosoftFoundry-e59cae5b.svg)AI 应用和智能体工厂

构建、优化和治理能够理解业务上下文并带来业务价值的 AI 应用和智能体，从未如此简单。我们统一、可互操作的 AI 平台帮助开发人员更快、更智能地开发，同时组织可在统一门户中实现全队安全和治理。

访问 11,000 多个基础模型、开放模型、推理模型、多模态模型和行业特定模型。针对你的用例即时对模型进行基准测试和比较，或启用智能模型路由以实时优化性能。

## 商业内超丰富的模型选择

## 使用开放、灵活的框架编排 AI 应用和智能体

通过最热门开源智能体框架的无缝托管与集成，简化开发流程。利用原生 IDE 体验和广泛的语言支持，开发人员可借助 Microsoft Agent Framework、LangChain、CrewAI 和 LlamaIndex 以 AI 速度推进开发。

![Microsoft Foundry 的示例智能体模板，其中显示了 GitHub Insights 和 Code Advisor 模板以及市场趋势研究助手模板。](https://ai.azure.com/assets/LightCarousel-2-fb814315.svg)

## 通过面向行动且具备上下文感知的智能体推动业务成果

轻松自定义和升级模型，赋能智能体理解业务上下文，并使用预构建和自定义 MCP 工具让 AI 在业务系统中执行操作。

![在 Microsoft Foundry 中配置智能体并将其连接到 MCP 和代码解释器等工具的示例。](https://ai.azure.com/assets/LightCarousel-3-58b1a3c0.svg)

## 通过集中式控制平面保护和管理 AI

在整个 AI 开发生命周期内实现全舰队监视和治理，以检测并缓解有害内容，同时通过丰富的 Microsoft Defender 和 Entra 集成，以迭代方式提升安全性、性能、成本和治理。

![Microsoft Foundry 监控示例图表，显示智能体的运行量随时间的变化情况。](https://ai.azure.com/assets/LightCarousel-4-312153fb.svg)
