# OpenClaw 背后核心框架 Pi：好的 Coding Agent 应该让用户来决定需要什么

Original Founder Park | Founder Park

OpenClaw 是当下最火的开源个人 AI 助手。很多人不知道的是，OpenClaw 背后核心是一个极简框架 Pi-coding-agent。

在 OpenClaw 的系统架构中，Pi agent 是 Gateway 控制层的核心子系统，控制了所有 agent 的推理和工具调用。

Pi 框架的设计哲学是让用户来决定自己需要什么样的 Coding Agent，而不是由框架预设一种固定的工作方式。这种灵活性使得 OpenClaw 能够适配各种不同的使用场景和工作流程。

Pi 的核心特点在于其极简的架构设计——用最少的代码实现最大的灵活性，让 Agent 能够根据用户的具体需求和偏好来调整自己的行为模式。
